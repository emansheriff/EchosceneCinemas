EchoScene is a dynamic cinema website designed to deliver an engaging movie-booking experience. The project focuses on visual appeal, smooth interactions, and realistic cinema flow. Inspired by fun moments that happen in real cinemas, EchoScene brings a fresh, playful identity to movie browsing.
Built using PHP, HTML/CSS/JavaScript, and MySQL, the website supports dynamic movie pages, showtime listings, and improved UI styling.
